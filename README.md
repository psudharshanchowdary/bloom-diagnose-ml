# Bloom Diagnose — AI Plant Disease Detection

AI-powered plant disease detection system. Upload an image of a crop leaf and get an instant diagnosis with treatment recommendations.

---

## What It Does

- Classifies plant diseases from leaf images using machine learning
- Covers common crops: tomato, potato, corn, grape, and more
- Returns disease name, confidence score, and treatment recommendations
- Web interface for easy image upload and results display

---

| Component | Technology |

## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.8+

### Installation

```bash
# Clone the repo
git clone https://github.com/psudharshanchowdary/bloom-diagnose-ml.git
cd bloom-diagnose-ml

# Install frontend dependencies
npm install

# Start the development server
npm run dev
```
